# folder-sync

Is an open source, Rust program for synchronizing folders.

Use as:

&nbsp;&nbsp;&nbsp;&nbsp;**folder-sync** *source* *destination* [-r]

Where:

&nbsp;&nbsp;&nbsp;&nbsp;*source* &nbsp;&nbsp;&nbsp;&nbsp;is the source folder.

&nbsp;&nbsp;&nbsp;&nbsp;*destination* &nbsp;&nbsp;&nbsp;&nbsp;is the destination folder.

&nbsp;&nbsp;&nbsp;&nbsp;-r&nbsp;&nbsp;&nbsp;&nbsp;is an optional flag to remove files in the destination folder that no longer exist in the source folder.
